# eslint-config

### In Node.js and Next.js

Install dependencies:
```
npm i -D @ogustavohp/eslint-config
```
Inside `.eslintrc.json`
```
{
  "extends": "@ogustavohp/eslint-config/next-node"
}
```
### In React and React Native

Install dependencies:
```
npm i -D @ogustavohp/eslint-config
```
Inside `.eslintrc.json`
```
{
  "extends": "@ogustavohp/eslint-config/react"
}
```
